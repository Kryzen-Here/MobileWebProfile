## 🚀 Run Locally

Clone the repository:

```bash
git clone https://github.com/yourusername/kryzen-portfolio.git

cd kryzen-portfolio
```

Open:

```text
index.html
```

Or use Python:

```bash
python -m http.server 8080
```

Then visit:

```text
http://localhost:8080
```

## 🎨 Customization

### Change Profile Picture

Replace:

```text
assets/profile.jpg
```

Recommended:

* 400×400 minimum
* Square image

### Change Name

Search:

```html
KRYZEN
```

Replace with your own name.

### Change Titles

Inside `main.js`:

```javascript
var titles = [
  "Embedded Architect",
  "AI Builder",
  "Digital Reaper",
  "Hardware Summoner",
  "Code Necromancer"
];
```

### Change Colors

Inside `style.css`:

```css
:root{
 --bg:#050505;
 --primary:#00ff88;
 --secondary:#00cc66;
 --white:#ffffff;
 --glow:#32ff9c;
}
```
